CRUD Driver System

This project was a simulation done to prepare us for the SAEP practical test (a test organized to measure the level of students in each SENAI course).

I used USBWebServer (to view the database in phpMyAdmin and view the system) in a VScode environment, and CodeIgniter as a framework, combined with Bootstrap, to develop this system.

Ilumination control

In this project, a web interface was created using nodejs and bootstrap. Both communicate, through an MQTT broker, with an esp32 that controls a dimmer connected to a lamp. With node, libraries such as express and mqtt were used to create a server for display and data traffic.

It was a project where I was able to not only improve my knowledge in node, but also develop my knowledge in esp32 and programming in the Arduino IDE environment.
